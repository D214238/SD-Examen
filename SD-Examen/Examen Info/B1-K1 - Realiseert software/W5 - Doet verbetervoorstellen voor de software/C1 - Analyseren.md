# Vereisten
___
- Maak een verbetervoorstel
- Vermeld bronnen bij je voorstellen zoals:
    - Teamgenoot als iemand of jijzelf bijvoorbeeld slechte code aanwijst of iets wilt verbeteren op technisch niveau.
    - Gebruikers of opdrachtgevers die dingen graag anders zien of extra functionaliteiten willen.
    - Testen die falen.

\* Voor een hogere beoordeling, laat aantoonbaar meer zien dan bovenstaande verwachtingen.
