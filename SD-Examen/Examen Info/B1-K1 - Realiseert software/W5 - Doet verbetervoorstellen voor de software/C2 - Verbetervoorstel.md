# Info
___
> [!section]- #### Verbetervoorstel ^verbetervoorstel
> ___
> Dit is een document met onderdelen die verbeterd of aangevuld kunnen worden in de volgende sprint. Een verbetervoorstel is meestal opgebouwd uit het volgende:
>
> - Het probleem of de huidige situatie.
> - Waarom het een probleem is of waarom het nuttig is.
> - De bron (categorie)
>   - Test
>   - Leidinggevende
>   - Teamgenoot
>   - Opdrachtgever
>   - Gebruiker
>   - Ontwerp
>   - …
> - Specifieke bron
> - Oplossing
> - Impact / prioriteit
> - Werkduur


# Minimale Vereisten
___
- Heeft de belangrijkste verbetervoorstellen maar zijn moeilijk realiseerbaar.
- Verbetervoorstellen komen niet natuurlijk voort uit het project / documentatie.

# Vereisten
___
- Heeft de belangrijkste verbetervoorstellen en zijn allemaal realiseerbaar.
- Verbetervoorstellen komen natuurlijk voort uit het project / documentatie.
